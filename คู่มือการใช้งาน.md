# คู่มือการใช้งาน superagent

เวอร์ชัน 0.1.34 · Minecraft Education 1.21.133

superagent คือ "ตัวละครผู้ช่วยพิเศษ" รูปก้อนสี่เหลี่ยม 1×1×1 ที่นักเรียนเขียนโปรแกรมควบคุมได้
(MakeCode บล็อก หรือ Python) เก่งกว่า Agent ปกติ — มีการรับรู้สภาพแวดล้อม, เดินอ้อมสิ่งกีดขวาง,
สร้าง/ขุด, จำข้อมูลข้ามเซสชัน, ต่อสู้, และพลังพิเศษ

ระบบมี **2 ส่วนที่ต้องติดตั้งคู่กัน**:
1. **Add-on (`.mcaddon`)** — ตัวละคร + สมองเบื้องหลัง (ลงในโลก)
2. **Extension** — บล็อก `superagent` ใน Code Builder (ลงในโปรเจกต์โค้ด)

> ⚠️ ต้องลงทั้ง 2 ส่วน ถ้าลงแค่อย่างเดียวจะใช้ไม่ครบ

---

## สิ่งที่ต้องเตรียม

- Minecraft Education **1.21.133** ขึ้นไป
- โลกที่ตั้งค่า: **Cheats = ON**, เกมโหมด Survival ได้, ผู้เล่นเป็น **Operator** (สำคัญ! หลายคำสั่งต้องใช้สิทธิ์นี้)
- ไฟล์ `dist/superagent-0.1.34.mcaddon`

---

## ส่วน A — ติดตั้ง Add-on (.mcaddon)

### A1. นำเข้า addon
1. ดับเบิลคลิกไฟล์ **`superagent-0.1.34.mcaddon`** → Minecraft Education จะ import ให้อัตโนมัติ
2. ถ้าดับเบิลคลิกไม่ได้ ให้เปิด Minecraft → **Settings → Storage → Import** แล้วเลือกไฟล์

### A2. เปิดใช้ในโลก
1. สร้างหรือแก้ไขโลก → ไปที่ **Behavior Packs**
2. หา **"superagent 0.1.34 Behavior"** → กด **Activate**
3. ไปที่ **Resource Packs** → เปิด **"superagent 0.1.34 Resources"** ด้วย
4. ยืนยันตั้งค่าโลก: **Cheats = ON**, Player Permissions = **Operator**

### A3. ตรวจว่าทำงาน
เข้าโลกแล้วดูในแชต ควรขึ้น:
```
superagent 0.1.34 script active
```
และจะเห็น **คิวบ์สีชมพู-ขาว** โผล่ใกล้ตัว มีชื่อ **`superagent [idle]`** ลอยเหนือหัว
ถ้าเห็นครบ = addon ทำงานแล้ว ✅

> ถ้าขึ้น "packs failed to load" หรือไม่เห็นข้อความ script active: ดูหัวข้อ **แก้ปัญหา** ด้านล่าง

### A4. อัปเดตเวอร์ชันใหม่
Minecraft มักไม่อัปเดต pack เวอร์ชันเดิม — เวลามีตัวใหม่ให้ **เอา pack เก่าออกจากโลกก่อน** แล้ว
**Activate ตัวใหม่** (อย่าแค่ import ทับ)

## ส่วน B — ติดตั้ง Extension (บล็อกใน Code Builder)

### B1. เปิด Code Builder
ในเกม กด **C** (หรือ Agent icon) → เลือก **MakeCode**

### B2. นำเข้า extension — เลือกวิธีใดวิธีหนึ่ง

**วิธี 1 — จาก GitHub (แนะนำสำหรับใช้ในห้องเรียน)**
1. สร้างโปรเจกต์ใหม่ → กดหมวด **Extensions** (ล่างสุดของกล่องบล็อก)
2. วาง URL นี้แล้ว import:
   ```
   https://github.com/numraise/raisesuperagent
   ```
   หรือแบบล็อกเวอร์ชัน: `https://github.com/numraise/raisesuperagent#superagent-0.1.34`

**วิธี 2 — วางโค้ด (เร็ว ไม่ต้องใช้อินเทอร์เน็ต)**
1. สร้างโปรเจกต์ใหม่ → สลับเป็นโหมด **JavaScript** (ปุ่มบนกลางจอ)
2. เปิดไฟล์ `superagent.ts` → คัดลอกทั้งหมด → วางทับในโปรเจกต์ → สลับกลับเป็น **Blocks**

### B3. ตรวจว่าทำงาน
จะเห็นหมวดสีม่วงชื่อ **SUPERAGENT** (ไอคอนหัวใจ) ในกล่องบล็อก ข้างใน มีกลุ่มย่อย
Control / Sensing / Thinking / Judging / Communicate / Navigation / Combat / Powers / Build /
Blueprint / Mine / Memory / Squad / Mission / Teacher / Events / Status

> ถ้าขึ้น "Extension Errors" ตอน import: กด **"Ignore errors and open"** แล้วเช็คว่าหมวด
> SUPERAGENT ใช้ได้ปกติ (มักเป็น cache ชั่วคราว)

### B4. อัปเดต extension
กดปุ่ม **refresh ⟳** ข้างชื่อ "superagent" ในแผง Explorer (ดึง commit ใหม่) — ถ้าไม่ขยับ ให้ลบ
extension แล้ว import URL ใหม่ (pin เวอร์ชัน) หรือใช้วิธีวางโค้ดทับ

## เริ่มใช้งาน — โปรแกรมแรก

```
on chat command "go"
    superagent spawn at player
    superagent walk to x 100 y 64 z 200
```
เข้าเกม พิมพ์ `go` ในแชต → ตัวละครจะมาอยู่ที่ตัวเรา แล้วเดินลื่นไปยังพิกัด (หยุดเมื่อชนกำแพง)

---

## คู่มือบล็อกทั้งหมด (จัดตามกลุ่ม)

### 🕹️ Control — ควบคุมการเคลื่อนที่
| บล็อก | ทำอะไร |
|---|---|
| `spawn at agent` / `spawn at player` | ให้ตัวละครไปอยู่ที่ Agent / ผู้เล่น |
| `recall to agent` | เรียกกลับมาที่ Agent |
| `move <ทิศ> blocks <n>` | เดินทีละช่อง **ชนกำแพงหยุด** (ทิศ: north/east/south/west/up/down) |
| `face <ทิศ>` | หันหน้าไปทิศที่กำหนด |
| `stop` | หยุดทุกการเคลื่อนที่ |
| `dash / scout / patrol / orbit / evade / high ground / zigzag / spiral` | รูปแบบการเคลื่อนที่สำเร็จรูป |

### 🧭 Navigation — เดินอัจฉริยะ
| บล็อก | ทำอะไร |
|---|---|
| `walk to x/y/z` | เดินลื่นเป็นเส้นตรง (ชนกำแพงหยุด) |
| `walk to agent` | เดินลื่นไปหา Agent |
| `follow walk <on/off>` | เดินตาม Agent ตลอด |
| `path to x/y/z` / `path to agent` | **เดินอ้อมสิ่งกีดขวาง (A\*)** หาเส้นทางเอง |
| `reached x/y/z` → boolean | ถึงจุดนั้นหรือยัง (รัศมี 2 บล็อก) |
| `walk stop` | หยุดเดิน |

### 👁️ Sensing — รับรู้สภาพแวดล้อม (คืนค่าให้ใช้ใน if/loop)
| บล็อก | คืนค่า |
|---|---|
| `hostiles within <r>` / `sense <mob> within <r>` | boolean — มีศัตรูในระยะไหม |
| `nearest hostile distance up to <max>` | number — ระยะถึงศัตรูใกล้สุด (-1=ไม่เจอ) |
| `nearest <block> distance/direction up to <max>` | number — หาบล็อก/แร่ |
| `block <ทิศ> of agent` | boolean — มีบล็อกด้านนั้นของ Agent ไหม |
| `path clear <ทิศ> of character` | boolean — ช่องนั้นว่างไหม |
| `distance to agent up to <max>` | number — ระยะถึง Agent |
| `ground below agent` | boolean — มีพื้นใต้ Agent ไหม |

### 🧠 Thinking — คิด/จำ (ข้อมูลอยู่ข้ามเซสชัน)
| บล็อก | ทำอะไร |
|---|---|
| `random 1 to <max>` | สุ่มเลข |
| `remember <key> = <value>` / `forget <key>` | เก็บ/ลบค่า |
| `memory <key> = <value>` / `>= <value>` | boolean — เทียบค่า |
| `memory <key> value up to <max>` | number — อ่านค่ากลับ |
| `count up <key>` | เพิ่มตัวนับทีละ 1 |
| `set flag <key> <on>` / `flag <key> is on` | ตั้ง/อ่าน ธงเปิดปิด |

### ⚖️ Judging — ตัดสินใจ
| บล็อก | คืนค่า |
|---|---|
| `should attack within <r>` | boolean — ควรโจมตีไหม |
| `is safe within <r>` | boolean — ปลอดภัยไหม |
| `danger close within <r>` | boolean — อันตรายประชิด (ศัตรู ≤ 3 บล็อก) |

### 💬 Communicate — สื่อสาร
| บล็อก | ทำอะไร |
|---|---|
| `report <text>` | แสดงข้อความบนจอผู้เล่น |
| `meet agent` | เดินอ้อมไปหา Agent |
| `label <text>` | ตั้งชื่อป้ายเหนือหัวตัวละคร |

### ⚔️ Combat — ต่อสู้ (ตัวละครหมุนตอนโจมตี)
| บล็อก | ทำอะไร |
|---|---|
| `attack from character radius <r> strength <s>` | โจมตีม็อบรอบตัว |
| `defend if hostiles within <r> strength <s>` | โจมตีเฉพาะเมื่อเจอศัตรู |
| `attack aura / guard agent / power burst / smart sweep / overdrive` | รูปแบบโจมตีต่าง ๆ |
| `auto guard <on/off>` | เปิด/ปิดโจมตีอัตโนมัติ (ค่าเริ่มต้น = ปิด) |

### ⚡ Powers — พลังพิเศษ
| บล็อก | ทำอะไร |
|---|---|
| `lightning strike` | ฟาดสายฟ้าลงศัตรูใกล้สุด |
| `force blast radius <r>` | ผลักม็อบกระเด็นออก |
| `shield player for <s> s` | เกราะกันตายให้ผู้เล่น |
| `heal player` | รักษาผู้เล่นเต็ม + regeneration |
| `magnet items radius <r>` | ดูดของที่ตกมาเข้าผู้เล่น |
| `blink player to character` | เทเลพอร์ตผู้เล่นมาหาตัวละคร |
| `summon ally for <s> s` | เรียก iron golem ชั่วคราวมาช่วยรบ |

### 🧱 Build / Blueprint — สร้าง
`build box/hollow box/floor/wall/pillar/pyramid/staircase/circle/disc`, `build row <pattern>`,
`build layer/blueprint <rows>` (สร้างจาก array ข้อความ), `clear area`, `copy region` + `paste here`,
`replace <from> with <to>` — เลือกชนิดบล็อกจากเมนู (stone/cobblestone/dirt/oak planks/glass/glowstone/sandstone/air)

### ⛏️ Mine — ขุด (ของเข้ากระเป๋า Agent)
`mine forward <length>`, `mine down <depth>`, `strip mine` — ต้องวาง Agent ให้ถูกจุดก่อน

### 🛡️ Squad — กองกำลัง
`summon guard` (เพิ่ม guard อัตโนมัติสูงสุด 4 ตัว วนล้อมผู้เล่น), `dismiss guards`

### 🏆 Mission — ภารกิจ/คะแนน
`mission start <title>`, `award <points>`, `score up to <max>`, `mission complete`,
`show scoreboard` (คะแนนอยู่ข้ามเซสชัน เอาไว้จัดอันดับห้องเรียน)

### 👩‍🏫 Teacher — ครูควบคุม
`freeze all` / `unfreeze all`, `gather all` (รวมตัวละครมาหาครู), `reset squad`,
`set home` / `go home` / `clear home`

### 📡 Events — เขียนแบบ reactive
`on superagent hostile within <r>`, `on area clear within <r>`, `on path blocked <ทิศ>`,
`watch on/off`, `check events`

## ตัวอย่างโปรแกรม

**1) ฮีโร่คุ้มกันผู้เล่น (Python)**
```python
superagent.spawn_at_player()
superagent.auto_guard(True)
while True:
    if superagent.should_attack(8):
        superagent.lightning_strike()
        superagent.force_blast(6)
        superagent.shield_player(15)
    elif superagent.is_safe(8):
        superagent.report("All clear")
```

**2) นักสำรวจเดินอ้อมกำแพง (Blocks)**
```
on chat command "explore"
    superagent spawn at player
    superagent path to x 150 y 70 z 150
    repeat until (superagent reached x 150 y 70 z 150)
        // รอถึงจุดหมาย
```

**3) สร้างบ้านจาก blueprint 3D (Python)**
```python
superagent.spawn_at_player()
superagent.build_blueprint(SuperagentBlock.OAK_PLANKS, [
    "XXX", "X.X", "XXX",   # ชั้นล่าง
    "-",                    # ขึ้นชั้นใหม่
    "XXX", "XXX", "XXX",   # หลังคา
])
```

## คำสั่ง /scriptevent (ใช้ได้โดยไม่ต้องมี extension — สำหรับครู/ทดสอบ)

พิมพ์ในแชตได้เลย (ต้องเป็น Operator):
```
/scriptevent superagent:combat on        เปิดโจมตีอัตโนมัติ
/scriptevent superagent:burst            โจมตีรอบตัวทันที
/scriptevent superagent:recall           เรียกตัวละครมาที่ผู้เล่น
/scriptevent superagent:freeze on        หยุดทุกตัวละคร
/scriptevent superagent:gather           รวมตัวละครมาหาครู
/scriptevent superagent:lightning        ฟาดสายฟ้า
```

## แก้ปัญหาที่พบบ่อย

| อาการ | สาเหตุ / วิธีแก้ |
|---|---|
| "packs failed to load" / ไม่ขึ้น script active | pack ไม่โหลด — ตรวจว่า activate ทั้ง BP+RP, เอาเวอร์ชันเก่าออกก่อนลงใหม่ |
| ไม่เห็นคิวบ์ตัวละคร | ยังไม่เปิด Resource pack หรือยังไม่ spawn — ลอง `spawn at player` |
| มีตัวละครหลายตัว | ตัวค้างจากเวอร์ชันเก่า — v0.1.30+ จะลบให้เหลือตัวเดียวอัตโนมัติภายในไม่กี่วินาที |
| บล็อกสั่งแล้วไม่เกิดอะไร | ผู้เล่นไม่ใช่ Operator / Cheats ปิด → เปิดในตั้งค่าโลก |
| Extension ขึ้น error สีแดง | type ผิด — ตรวจว่า import เวอร์ชันตรงกับ addon, หรือ refresh/วางโค้ดใหม่ |
| ตัวละครทะลุกำแพง | ใช้ `walk to`/`path to`/`move` (v0.1.31+ มี collision) ไม่ใช่บล็อกเก่า |
| มี effect เยอะ/รก | v0.1.31+ เอา particle ไอเดิลออกแล้ว — อัปเดตเป็นเวอร์ชันล่าสุด |
