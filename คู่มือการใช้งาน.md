# คู่มือการใช้งาน superagent

เวอร์ชัน 0.1.56 · Minecraft Education 1.21.133

superagent คือตัวละครผู้ช่วยพิเศษแบบ special mob 1×1×1 ที่นักเรียนควบคุมผ่าน MakeCode ได้ทั้ง Blocks/Python
(เร็วขึ้นกว่า Agent เดิมและมีพฤติกรรมแบบขุด-สร้าง-รับรู้-รบได้มากขึ้น)

ระบบมี 2 ส่วนที่ต้องติดตั้งคู่กัน:
1. **Add-on (`.mcaddon`)** — สคริปต์และพฤติกรรมตัว agent ในโลก
2. **Extension** — บล็อก `superagent` ใน Code Builder

> หมายเหตุ: ไม่ต้อง push `.mcaddon` ขึ้น GitHub/repo และไม่ต้องแนบเป็น release asset
> ให้เก็บเป็นไฟล์ local เท่านั้น (`dist/superagent-0.1.56.mcaddon`)

---

## สิ่งที่ต้องเตรียม

- Minecraft Education 1.21.133+ 
- Cheats = ON
- เกมโหมด Survival ทำงานได้ปกติ แต่สำหรับบล็อกบางอย่างควรเป็น **Member/Operator** ตามข้อจำกัดของเกม
  - ในแนวปฏิบัติ: ทำงานใน **Member survival** ได้มากขึ้นตามที่ออกแบบไว้
  - คำสั่งที่เป็น server action บางคำสั่งอาจยังต้องสิทธิ์สูงกว่า
- สร้างไฟล์ addon แล้วนำเข้าเกม: `dist/superagent-0.1.56.mcaddon`

## A) ติดตั้ง Add-on

### A1. import addon
1. ดับเบิลคลิก `superagent-0.1.56.mcaddon` (หรือ import จาก Storage)
2. เข้าโลกที่ต้องการและเปิด Behavior + Resource Pack ของ superagent

### A2. เปิดใช้
1. สร้าง/แก้ไขโลก → Behavior Packs: เปิด **superagent 0.1.56 Behavior**
2. Resource Packs: เปิด **superagent 0.1.56 Resources**
3. ยืนยัน cheats ON และ permission อย่างน้อย Member

### A3. เช็กว่าทำงาน
- ในแชตควรเห็นข้อความ `superagent 0.1.56 script active`
- ขณะ spawn จะเห็น agent cube สีชมพู-ขาวพร้อมป้ายแสดงชื่อ

### A4. อัปเดตเวอร์ชัน
เมื่อมีเวอร์ชันใหม่ ให้ import และ activate ใหม่อีกครั้ง และเอา pack เก่าที่ยังค้างออกก่อนเพื่อหลีกเลี่ยงตัวซ้ำ

```

```
หรือ pin เวอร์ชัน

```
https://github.com/numraise/raisesuperagent#superagent-0.1.56
```

### B3. ตรวจว่าปรากฏหมวด
ควรเห็นหมวด **SUPERAGENT** ใน toolbox พร้อมกลุ่ม
Control / Sensing / Thinking / Judging / Communicate / Navigation / Combat / Powers / Build / Blueprint / Mine / Memory / Mission / Events / Status

### B4. ถ้าไม่อัปเดต
กด refresh extension หรือ remove แล้ว import ใหม่ (ถ้าหน่วงหน่วยความจำใน browser cache)

## เริ่มต้นใช้งานเร็ว

```python
# Python
superagent.spawn_at_player()
superagent.label("TEST-001")
superagent.walk_to(-16, 68, 200)
superagent.path_to(-20, 68, 205)
superagent.report("ready")
```

```text
on chat command "go"
    superagent spawn at player
    superagent label "GO"
    superagent walk to x -16 y 68 z 200
```

## หมวดบล็อกหลัก (สรุปใช้งานจริง)

### การควบคุมและเคลื่อนที่
- `spawn at player` / `spawn at agent`
- `recall to agent`
- `move <direction> blocks <n>`
- `face <direction>`
- `walk to x y z`
- `path to x y z`
- `stop`

### การรับรู้สภาพแวดล้อม
- `hostiles within <r>`
- `sense <mob> within <r>`
- `nearest hostile distance up to <max>`
- `distance to agent up to <max>`
- `path clear <direction> of character`

### สื่อสาร / แสดงผล
- `report <text>` และ `report <value>` (รองรับตัวเลข/Boolean ได้)
- `label <text>` และ `label <value>` (รองรับค่าจากบล็อก reporter ได้)

### ตำแหน่งและทิศทาง
- `world position x` / `world position y` / `world position z` (ปัดเศษทศนิยมเป็นจำนวนเต็ม)
- `world position text`
- `world direction`
- `label world position`
- `report world position`

### ความจำและการตัดสินใจ
- `remember`, `memory`, `count up`, `flag`
- `should attack within`, `is safe within`, `danger close within`

### ต่อสู้และกำลัง
- `attack from character radius <r> strength <s>`
- `defend if hostiles within <r> strength <s>`
- `auto guard on/off`
- `lightning strike`, `force blast`, `shield player`, `heal player`, `magnet items`

### สร้าง/ขุด/จัดการโลก
- `build floor / wall / pillar / row / pyramid / staircase / clear area`
- `build blue print`, `mine forward`, `mine down`, `strip mine`, `copy/paste`

### เคลื่อนไหวตัวแทนผู้เล่น
- `teleport player to character`, `blink player to character`

## ติวให้เห็นภาพว่าใส่ค่าใน block ได้แบบ report/label

ตัวอย่าง block สายรายงานตำแหน่ง:

```text
on start
    superagent spawn at player
    superagent report superagent world z
    superagent report superagent world position text
    superagent label (superagent world x)
```

ผลลัพธ์ที่เห็น:
- `report` แสดงตัวเลข/ข้อความที่อ่านได้ในแชต
- `label` แสดงค่าเหนือหัวอัตโนมัติ
- สุ่มตัวเลขเป็น integer (ไม่มีเศษทศนิยม) ตามตำแหน่งเชิงโลก

## คำสั่ง /scriptevent (ไม่ต้องผ่าน extension)

```
/scriptevent superagent:combat on
/scriptevent superagent:combat off
/scriptevent superagent:burst
/scriptevent superagent:recall
/scriptevent superagent:freeze on/off
```

> คำสั่งแชตนี้ขึ้นอยู่กับสิทธิ์เกมและโลกที่ใช้งาน

## แก้ปัญหาที่พบบ่อย

- ไม่เห็นคิวบ์: ตรวจ add-on เปิดครบ BP/RP และเรียก `spawn at player`
- `report`/`label` ไม่แสดง: ตรวจสิทธิ์ผู้เล่น + ค่าถูกส่งเข้าบล็อกหรือยัง
- คำสั่งบางอันไม่ทำงาน: สลับ import แบบ pin เวอร์ชัน (`#superagent-0.1.56`) ใหม่
- ตัวละครเจอของผนัง: ใช้ `walk to`/`path to` แทนการคาดคำนวณทิศทางเอง
- หากเห็น pack ซ้อนหลายตัว: เอาแพคเวอร์ชันเก่าออกแล้วเปิดใหม่
